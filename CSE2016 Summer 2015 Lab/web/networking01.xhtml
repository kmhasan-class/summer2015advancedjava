Networking 01

Write a Java Client/Server calculator application that does as follows:

Client side

  • Connects to the server that is running on your computer at 2215
  • Once connection is established, it sends calculation requests in the form:
    • ADD x y
    • SUB x y
    • MUL x y
    • DIV x y
    • where x and y are 32 bit integers
  • Prints out the response it got from the server

Server side

  • Creates a socket and listens for client on port 2215
  • When the client sends a calculation request it sends back the result, for example:
    • Client sends: ADD 4 -5; Server responds: -1
    • Client sends: DIV 10 3; Server responds: 3
