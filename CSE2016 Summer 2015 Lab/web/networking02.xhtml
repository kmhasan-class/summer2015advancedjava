Networking 02

Write a Java Client/Server calculator application that does as follows:

Client side

  • Two different clients (say Client A and Client B) connect to the server that is running on your computer at 4200
  • Maintains their own names through a String data member
  • Client A sends a message
  • Client B sends a message
  • Both clients print out their own names followed by the message they got as response from the server

Server side

  • Creates a socket and listens for client on port 4200
  • Accepts message from the first client and responds: "Thank you!"
  • Accepts message from the second client and responds: "The first guy tells me X"
    • where X is the message it got from the first client
