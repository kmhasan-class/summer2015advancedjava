Drawing 01

Write a JavaFX application that draws a figure as described below:
  • Draw a circle with radius R at the center of the application
      Value of R will be provided when you are running the code
  • Draw a diameter line parallel to x-axis inside the circle
  • Draw N line segments (including the diameter) parallel to this diameter
    • Value of N will be provided to you when you are ready to run the code
    • You can assume that value of N will be odd
  • These N line segments need to
    • be totally contained inside the circle
    • touch the perimeter of the circle with both the ends of the line segments
    • be equally spaced along y-axis
