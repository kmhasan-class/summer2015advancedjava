File I/O 01

Write a JavaFX/File I/O application that does as follows:
  • Build a JavaFX UI that allows users to write some numbers in a TextField
  • The numbers in that TextField will be separated by spaces
  • Add a Sum button in that UI
  • When a user presses the Sum button the summation of all the numbers in the TextField will be written out to summation.txt file
  • You need to ensure that you're not deleting the contents of the summation.txt file, you'll be adding the new result at the bottom of the existing file
